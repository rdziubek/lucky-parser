Pomoc

  • <- wskazuje miejsce kolejnego tagu,
  • <html język kodowanie tytuł> tworzy tag html,
  • <img nazwa szerokość wysokość "tekst alternatywny"> tworzy tag img,
  • <data> wstawia bieżący czas,
  • <formularz akcja metoda input{tytuł typ}…> tworzy tag form,
  • <lista typ {treść…}> tworzy tag ul/ol,
  • <wideo szerokość wysokość źródło atrybut> tworzy tag video,
  • <link url tytuł> tworzy tag a,

Każdy element przyjmuje też atrybuty styl="" oraz .klasa-css wymieniane po tych domyślnych.

Oznaczenie … określa możliwość powtarzania argumentu poprzez poprzedzenie go spacją.
Wartości tekstowe mogą umieszczane być w cudzysłowach.
